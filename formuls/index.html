Справка для написания формул:

Для написания кол-ва молекул/ионов надо писать число ПЕРЕД элементом.
Пример: 2HCl

Для написания кол-ва атомов в элементе надо писать число После названия атома.
Пример: H2O

Для написания заряда атома после названия надо написать символ "/" и заряд атома (допускаетя символы "+" и "-")
Пример H/+Cl/-1
H2/+O/2/-
